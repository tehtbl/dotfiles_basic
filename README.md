tbl_dotfiles_basic
==================

This repo houses my basic dotfiles which are essential to me, e.g. Bash, Zsh, ...

Requirements
============

The system these dotfiles needs support for working with:

* Makefile
* git

How To Use
==========

Simply type: `make help`
